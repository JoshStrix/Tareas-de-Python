Instalar Visual Studio Code:

1.Si aún no tienes Visual Studio Code instalado, descárgalo e instálalo desde code.visualstudio.com.

2.Instalar Python:

Asegúrate de tener la ultima versionn de Python instalada en tu sistema. Puedes descargarlo desde python.org.
Abrir Visual Studio Code:

3.Inicia Visual Studio Code.

4.Abrir la Carpeta del Proyecto:

Haz clic en File (Archivo) en la barra de menú y selecciona Open Folder... (Abrir Carpeta...).
Navega hasta la carpeta donde se encuentra el archivo Examen.py y selecciónala.

5.Abrir el Terminal Integrado:

Haz clic en Terminal en la barra de menú y selecciona New Terminal (Nueva Terminal). Esto abrirá un terminal en la parte inferior de la ventana de VS Code.

6.Ejecutar el Código:

Asegúrate de que el terminal esté en el directorio correcto (donde se encuentra Examen.py). Si no lo está, puedes usar el comando cd para cambiar de directorio.
Escribe el siguiente comando en el terminal y presiona Enter:

python Examen.py

7.Ver los Resultados:

Después de ejecutar el comando, deberías ver la salida del programa en el terminal, mostrando la información de los artistas, sus obras y las exposiciones.
